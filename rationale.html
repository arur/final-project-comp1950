Rationale - Team AKA

Through a thorough analysis of the clients and end-users needs, we were able to identify the top three goals for the COMP1950 website:

  1. Ability to use lecture notes as an effective study aid
  2. Ability to easily access in-class exercises
  3. Ability to create an online learning environment that is supportive and intuitive

Current Issues

With the above goals kept in mind, we analyzed the current COMP1950 website and identified the following issues.

Difficult to Find Information

The current website has one long page for each of it's eleven lectures. While the student is able to go to the lecture of their choice via the right sidebar, the content structure of each lecture layout page proves to be difficult to find information, access exercises, and view examples. The student is constantly scrolling up and down each lecture page or using the ctrl+f option in order to search for specific information. This proves to be a less effective learning experience, as the student spends more time searching for information rather than digesting information.

Very Text Heavy

The current website is very text heavy, which in addition with the long page scroll, creates an overwhelming experience for the student. The headings used is the same or similar size to the body copy, and in general, there is a lack of headings used throughout the entire website. This decreases usability and visual consistency as it does not allow students to easily scan the content.

Not Visually Appealing

The current website uses three main colours in it's colour scheme: maroon, light and dark grey, and black. The current colour scheme is dull and uninspiring. This differs from the client's request of wanting the website to be visually appealing and enthusiastic yet professional.

Our Solution

Multi-Tier Navigation

We propose a 'multi-tier' navigation which includes a horizontal main menu and a left sidebar menu. The main menu portrays the websites parent pages while while the sidebar menu portrays the child pages or child content of the respective parent page. With the sidebar, students can easily locate information within that page by clicking on the submenu item, which will bring the student to the respective content. The addition of the sidebar also prevents from having one deep nested navigation.

Sectioned Lecture Submenus

Each lecture will be divided into different sections, with each section appearing as a submenu item on the sidebar. Students will be able to expand each lecture, and navigate to their preferred section of the lecture by clicking on the respective submenu item. Other than viewing the course contents, the student will be able to easily navigate to the exercises, homework assignments, and learning objectives.

Seperate Exercise Section

Students can now easily find the lecture exercises through the 'exercise section' which will appear within each lecture's submenu. Not only does this allow the student easier access to the exercises, but also allows the student to view the number and type of exercises at a glance. The exercises will appear on the same page as it's respective lecture counterpart rather than a seperate parent page. This is important as it allows the student to follow the flow of the lecture and to be able to easily associate the exercise with relevant parts of the lecture.

Searchable Content

With the addition of a search bar in the header, the student can now easily find information using keywords without having to dig into each lecture page.

Student Forum

Being able to engage and interact with your classmates and professor is just as important outside of the classroom. Our proposed student form now only allows the student to ask questions but also creates a supportive online learning environment. The addition of a forum allows students and their professor to discuss topics such as homework assignements, lecture notes, or current website trends.

Captivating Homepage

As the main landing page, the homepage should create a first good impression and be able to tell the viewer what the website is used for. Our proposed homepage is complete with a compelling image and effective call to action. Upon landing on the homepage, the viewer will be able to easily understand the website goals.
